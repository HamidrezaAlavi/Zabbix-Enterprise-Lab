# آزمایشگاه مانیتورینگ شبکه (Mini NOC) 📊

سلام! به ریپازیتوری آزمایشگاه شبکه من خوش اومدید. 

من همیشه دلم می‌خواست فراتر از تئوری‌های کتاب، واقعاً ببینم توی یک اتاق مانیتورینگ (NOC) چه اتفاقی می‌افته. به همین خاطر این لابراتوار رو راه انداختم تا زبیکس رو با تجهیزات واقعی و مجازی درگیر کنم.

## 🛠️ معماری لابراتوار من

برای این پروژه سعی کردم محیط مجازی رو به شبکه فیزیکی متصل کنم تا سناریو واقعی‌تر باشه:
* **هسته مانیتورینگ:** اوبونتو سرور که Zabbix 7.0 روش نصبه (روی VMware بالا آوردمش).
* **روتر اصلی:** یک ماشین مجازی Cisco CSR1000v با سیستم‌عامل IOS-XE که نقش روتر مرکزی رو بازی می‌کنه.
* **تجهیزات فیزیکی:** مودم اینترنت خودم (TP-Link TD-W8961N) برای مانیتور ترافیک واقعی.

## 🚀 کارایی که تو این پروژه انجام دادم

هدفم این بود که فقط به یک Ping ساده قناعت نکنم:

* **کانفیگ SNMP:** پروتکل SNMPv2 رو روی روتر سیسکو و مودم تی‌پی‌لینک فعال کردم تا دیتای لحظه‌ای پورت‌ها رو بکشم بیرون.
* **کشف خودکار (LLD):** با استفاده از تمپلیت‌های زبیکس کاری کردم که سیستم خودش پورت‌های شبکه، وضعیت CPU و رم رو پیدا کنه و براشون گراف بسازه.
* **مانیتورینگ وب (Web Scenarios):** برای تست سرویس‌ها، سناریوهایی نوشتم که زبیکس مثل یک کاربر واقعی سایت‌ها رو باز کنه، سرعت لود رو بسنجه و کد HTTP 200 رو چک کنه.
* **نقشه زنده شبکه:** یک توپولوژی (Network Map) طراحی کردم که اگر پورتی قطع شد، همون لحظه روی نقشه قرمز بشه و آلارم بده.
* **ربات هشدار در پیام‌رسان بله (Alerting):** با استفاده از Webhook و کانفیگ Trigger/Actionها، سیستم رو طوری تنظیم کردم که به محض قطعی تجهیزات یا بروز مشکل، زبیکس مستقیماً از طریق یک ربات در پیام‌رسان "بله" روی گوشی به من پیام هشدار بفرسته.
* **مدیریت دیتابیس:** قابلیت Housekeeping رو فعال کردم تا دیتای خامِ قدیمی پاک بشه ولی میانگین‌ها (Trends) برای گزارش‌گیری بمونه تا هارد سرور پر نشه.
* **مدیریت خاموشی‌ها (Maintenance Mode):** زمان‌هایی که می‌خواستم روترها رو ری‌استارت کنم یا تغییری بدم، سیستم رو می‌بردم رو حالت تعمیرات تا نصفه‌شب آلارم قطعی نفرسته!

## 📂 تو این ریپازیتوری چی پیدا می‌کنید؟
* `/dashboards`: اسکرین‌شات‌هایی که از داشبورد زنده، نقشه شبکه و گراف‌های ترافیک گرفتم.
* `/configs`: فایل‌های متنی شامل دستورات کامندلاینی (CLI) که روی روتر سیسکو زدم.
* `/templates`: فایل‌های بک‌آپ و تمپلیت‌هایی که تو این سناریو استفاده کردم.
